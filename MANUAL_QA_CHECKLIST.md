# Family News 수동 점검 체크리스트

## 1) 사진 업로드 화면
- [ ] 여러 장 이미지를 선택하면 미리보기가 모두 표시된다.
- [ ] 각 미리보기에서 `대표사진` 체크박스가 보인다.
- [ ] 체크박스는 한 장만 선택된다(다른 체크 자동 해제).
- [ ] 대표사진을 체크하지 않고 업로드하면 오류 메시지가 표시된다.
- [ ] 대표사진을 체크하고 업로드하면 상세 페이지 대표 이미지로 반영된다.
- [ ] 체크하지 않은 나머지 사진은 상세 페이지 슬라이드에서 다음 장으로 보인다.

## 2) 기사 수정 화면
- [ ] 현재 대표사진이 표시되고 `대표 사진 삭제` 체크박스가 보인다.
- [ ] 기존 추가사진마다 `삭제` 체크박스가 보인다.
- [ ] 기존 추가사진 삭제 체크 후 저장하면 해당 이미지가 삭제된다.
- [ ] 새 사진 업로드 시 미리보기별 `대표사진` 체크박스가 보인다.
- [ ] 새 사진 업로드 시 대표 체크를 안 하면 오류가 표시된다.
- [ ] 새 사진 대표 체크 후 저장하면 대표 이미지가 선택한 사진으로 교체된다.

## 3) 상세 페이지 슬라이드
- [ ] 한 번에 사진 1장만 보인다.
- [ ] 이전/다음 버튼으로 사진이 이동한다.
- [ ] 하단 도트 클릭 시 해당 사진으로 이동한다.
- [ ] 이미지가 너무 크게 넘치지 않고 카드 중앙에서 적절한 크기로 표시된다.

## 4) 기본 안정성
- [ ] 홈 → 상세 → 수정 → 저장 이동이 정상 동작한다.
- [ ] 권한 없는 사용자가 다른 사람 글 수정 URL 접근 시 상세로 리다이렉트된다.
- [ ] 서버 콘솔에 저장 시 치명 오류(traceback)가 없다.

## 확인 환경
- 서버: `python manage.py runserver --settings=config.settings.local`
- 접속: `http://0.0.0.0:8090/`
